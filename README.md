# 📁 Python File Explorer

A command-line File Explorer built with Python that allows users to perform common file and folder management operations directly from the terminal.

The project uses Python's built-in `os` module to interact with the operating system and provides a simple interface for navigating and managing files.

---

## Features

- 📂 Show current working directory
- 📋 List files and folders
- ✅ Check if a file or folder exists
- 📁 Create folders
- 🗑️ Delete empty folders
- ✏️ Rename files and folders
- 📄 Create files
- ❌ Delete files
- ⚠️ Error handling for common file system operations

---

## Technologies Used

- Python 3
- os module

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/TheAbdullahSaleem/python-file-explorer.git
```

Run the program:

```bash
python main.py
```
